# Each project is 100% self-contained:
#   - its own crew.py (domain agents + RSS feeds)
#   - its own SQLite dedup DB
#   - its own brand.py (colours, wordmark, section taxonomy)
#   - its own templates/ stub (extends the shared hikmah-shared/templates/base.html.j2)
#   - its own config/.env (separate ISSUE_NUMBER counter)
#   - its own output/ and logs/
#
# Every run emits SEPARATE files per device:
#   output/issue_NNN_desktop.html   output/issue_NNN_desktop.pdf   (A4)
#   output/issue_NNN_mobile.html    output/issue_NNN_mobile.pdf    (phone)
#
# Shared code lives in hikmah-shared/ and is imported by all five.
#
# RUN:
#   AUTO    python run_all.py                 (5 weekly schedulers, blocks)
#           python hikmah-<proj>/scheduler.py (one project's weekly cron)
#   MANUAL  python run_now.py <proj|all>      (on demand: crew->render->publish->email)
#   PREVIEW python run_now.py <proj|all> --demo   (offline, no API key)
